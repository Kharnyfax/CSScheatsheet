CSS Selector Reference

Selector Example Description
Element p Selects all < p > elements
Class .banner Selects all elements with class="banner"
Id #button Selects all elements with id="button"
* * Selects all elements
Element.class p.banner Selects all < p > elements with class="banner"
Element, element p, span Selects all < p > elements and < span > elements
Element element span p Selects all < span > elements inside < p > element
